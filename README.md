# 👤 Customer Persona Generator – AI-Powered Marketing Persona Application

## 📌 Project Overview
Customer Persona Generator is an AI-powered web application designed to create highly detailed and accurate marketing personas from simple user prompts. It helps marketers, business owners, content creators, and product teams generate data-driven customer profiles quickly using Artificial Intelligence, enabling them to better understand and target their ideal audience.

## ✨ Features
- 🤖 AI-Generated Personas: Leverage powerful AI to craft in-depth customer avatars and buyer personas.
- 📝 Detailed Customer Profiles: Generates comprehensive details including demographics, pain points, goals, and buying behaviors.
- 🎨 Clean UI: A simple, intuitive, and user-friendly web interface.
- ⚡️ Fast Execution: Rapid response generation for quick marketing strategy planning.
- 🔒 Secure API Management: Safely manages sensitive API keys using .env files.
- 🎯 Targeted Marketing: Perfect for ad campaigns, product development, and content strategy.

## 🛠 Technologies Used
* Backend: Python, Flask
* Frontend: HTML5, CSS3, JavaScript
* AI Integration: Google AI Studio
* Tools: Git, GitHub, Visual Studio Code
