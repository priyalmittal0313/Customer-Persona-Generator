CaptionAI – AI Social Media Caption Generator
📌 Project Overview
CaptionAI is an AI-powered web application that generates engaging and creative social media captions from user prompts. It helps content creators, businesses, marketers, and individuals generate creative, engaging, and platform-specific captions quickly using Artificial Intelligence.

✨ Features
🤖 AI-generated social media captions
📝 Generates creative and engaging content
🎨 Clean and user-friendly interface
⚡️ Fast response generation
🔒 Secure API key management using .env
🌐 Built with Flask and Python
💡 Easy to use for Instagram, Facebook, LinkedIn, Twitter (X), and other social media platforms
🛠 Technologies Used
Python
Flask
HTML5
CSS3
JavaScript
Groq API
Git & GitHub
Visual Studio Code
